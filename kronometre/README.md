# Kronometre Uygulaması 

Kullanıcının kronometreyi başlatabildiği, durdurabildiği ve sıfırlayabildiği basit bir zaman ölçer uygulamasıdır.

## Özellikler

* Kronometre başlatma
* Kronometre durdurma
* Kronometre sıfırlama
* Saat / dakika / saniye takibi
* 00:00:00 formatında gösterim


##  Canlı Demo
https://eliife.github.io/mini-project/kronometre


## Notlar
* Aynı anda birden fazla interval çalışması engellenmiştir
* Sıfırlama işlemi kronometreyi tamamen başa döndürür
* Proje öğrenme amaçlı geliştirilmiştir
