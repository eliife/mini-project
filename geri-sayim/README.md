# Geri Sayım Sayacı 
Kullanıcının seçtiği tarihe kalan süreyi anlık olarak gösteren basit bir geri sayım uygulaması.

##  Özellikler
* Tarih seçme
* Geri sayımı başlatma
* Gün / saat / dakika / saniye olarak gösterme
* Süre dolunca uyarı verme


##  Canlı Demo
 https://eliife.github.io/mini-project/geri-sayim

##  Not
* Geçmiş bir tarih seçilirse uyarı verir.
* Tarih seçilmeden başlatılamaz.
