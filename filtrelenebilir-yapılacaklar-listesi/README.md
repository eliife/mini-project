# Filtrelenebilir Yapılacaklar Listesi

Kullanıcının görev ekleyebildiği, tamamlayabildiği ve filtreleyebildiği basit bir yapılacaklar listesi uygulaması.

## Özellikler

* Görev ekleme
* Görevleri tamamlandı olarak işaretleme
* Tamamlanan / tamamlanmayan görevleri filtreleme
* Enter tuşu ile hızlı ekleme


##  Canlı Demo
https://eliife.github.io/mini-project/filtrelenebilir-yapılacaklar-listesi


## Not
* Boş görev eklenemez
* Filtre seçimine göre liste güncellenir
