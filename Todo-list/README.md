## To-Do List
Bu proje, HTML, CSS ve JavaScript kullanılarak geliştirilmiş modern ve işlevsel bir yapılacaklar listesidir.

## Özellikler
-Görev ekleme, düzenleme ve silme
-Kategori (İş, Okul, Kişisel) seçimi
-Son tarih (Deadline) belirleme
-Tarayıcı hafızası (LocalStorage) ile verileri saklama
