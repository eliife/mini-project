# Form Doğrulama Uygulaması 

Kullanıcının ad ve e-posta bilgilerini kontrol eden basit bir form doğrulama uygulaması.

## Özellikler

* Ad alanı boş kontrolü
* E-posta format kontrolü
* Hatalı girişlerde uyarı mesajı
* Doğru girişte başarı mesajı


## Canlı Demo
 https://eliife.github.io/mini-project/formDogrulama


##  Not
* Sayfa yenilenmeden kontrol yapılır
* E-posta kontrolü için Regex kullanılmıştır
